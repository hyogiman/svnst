# 柒 The Seven Statements — 배포 & 운영 가이드

일곱 개의 진술: 진실 4 + 거짓 3, 심문과 배팅으로 서로를 알아가는 조직문화 게임.

## 파일 구성

| 파일 | 역할 |
|---|---|
| `index.html` | 참가자 앱 (작가의 방 → 게임 → 결과) |
| `admin.html` | 관리자 콘솔 (단계 제어·모니터링·공지·결과 집계) |
| `firebase-config.js` | Firebase 프로젝트 설정 (두 페이지가 공유) |
| `firestore.rules` | Firestore 보안 규칙 |

## 1. Firebase 설정 (최초 1회, 약 15분)

1. **프로젝트 생성** — [Firebase 콘솔](https://console.firebase.google.com) > 프로젝트 추가
2. **Firestore 활성화** — 빌드 > Firestore Database > 데이터베이스 만들기 (프로덕션 모드, 위치 asia-northeast3 서울)
3. **익명 인증 켜기** — 빌드 > Authentication > 시작하기 > Sign-in method > **익명** > 사용 설정 ✅
4. **웹 앱 등록** — 프로젝트 설정(⚙️) > 일반 > 내 앱 > 웹(</>) 추가 → 표시되는 `firebaseConfig` 값을 `firebase-config.js`에 붙여넣기
5. **보안 규칙 배포** — Firestore Database > 규칙 탭 > `firestore.rules` 내용 전체를 붙여넣고 게시

## 2. 관리자 등록 (최초 1회)

보안 규칙상 `config` 쓰기(게임 시작/종료)는 **admins 컬렉션에 등록된 UID**만 가능합니다.

1. 관리자로 쓸 브라우저에서 `admin.html` 접속 → 게이트 화면 하단에 **내 UID** 표시됨
2. Firebase 콘솔 > Firestore > **`admins` 컬렉션 생성** > 그 UID를 **문서 ID**로 빈 문서 추가
3. `admin.html` 상단의 `ADMIN_PW` 상수를 원하는 비밀번호로 변경 (화면 잠금용)

> ⚠️ 익명 인증 UID는 브라우저별로 발급됩니다. 브라우저 데이터를 지우거나 다른 기기를 쓰면
> UID가 바뀌므로 admins에 다시 등록해야 합니다. 행사 당일 쓸 노트북/폰을 미리 정해두세요.

## 3. 배포 (GitHub Pages)

1. 저장소에 4개 파일 업로드
2. Settings > Pages > Branch: `main` > Save
3. `https://아이디.github.io/저장소명/` = 참가자 주소 (QR로 배포)
4. `https://아이디.github.io/저장소명/admin.html` = 관리자 주소

## 4. 당일 운영 순서

1. **[관리자]** 설정 확인(행사 코드·초기 코인 100·티켓 2장·배팅 대기 60초) → **① 작가의 방 열기**
2. **[참가자]** QR 접속 → 행사 코드 + 이름/소속 입장 → 일곱 문장 작성 → 준비 완료
   - 관리자 화면에서 작성 완성률(n/7) 실시간 확인
3. **[관리자]** 대부분 준비되면 → **② 게임 시작** → 전원 화면이 자동으로 게임 모드 전환
   - 미완성자는 작성을 마치는 즉시 자동 합류
4. 진행 중: 공지 발송, 참가자 제외/부활, 랭킹 모니터링
5. **[관리자]** **③ 게임 종료 & 결과 공개** → 참가자 전원에게 개인 결산 + TOP 5 표시
   - **'믿기 힘든 진실상' 집계하기** 버튼 → 클로징 세리머니에서 발표

## 진행자 안내 멘트에 넣을 것 (앱에는 없는 규칙)

- **"여러분의 거짓말 라이선스는 딱 세 문장뿐입니다.** 진실에 대한 답은 아껴도 되지만, 지어내지는 마세요."
- 상대를 만나면 먼저 인사하고, 문장에 대해 자유롭게 묻고 답하세요. 배팅은 대화가 충분할 때.
- 공개 후에는 서로의 진짜 뒷이야기를 나누고 헤어지세요.

## 게임 규칙 요약

- 초기 100코인 / 배팅 🔥30 · 🤔20 · 🎲10
- 추리 적중 시 배팅액 획득, 실패 시 상대에게 지급
- 잔고 10 미만이면 10으로 충전 (찍기 배팅만 가능한 재기 지원선)
- 리라이트 티켓 2장: 거짓 문장만, 만남 중 아닐 때만, 1회 20코인
- 🕵️ 명탐정 랭킹 = 추리로 딴 누적액 / 🎭 명배우 랭킹 = 방어로 뺏은 누적액
- 한 번 만난 상대와는 재매칭 불가

## 알아두면 좋은 것 / 한계

- **재접속**: 같은 폰·같은 브라우저면 자동 복귀(익명 인증 지속성). 기기를 바꾸면 새 참가자로 인식됩니다.
- **정답 보안**: 거짓 문장 정보(`answers`)는 본인만 읽을 수 있어 개발자 도구로도 남의 정답을 볼 수 없습니다.
- **만남 중단**: 상대가 자리를 떠나면 만남 화면 하단의 [만남 취소]로 빠져나올 수 있습니다(해당 만남은 무정산).
- **행사 초기화**: 다음 행사 전 Firestore에서 `participants` `statements` `answers` `meetings` `betlog` 컬렉션을 삭제하세요. (config와 admins는 유지)
